# fusion_api_flutter

A flutter wrapper of [Fusion Cloud API](https://datameshgroup.github.io/fusion/docs/getting-started).

## Instruction

The usage is simple:

1. Pairing POI with POS
  a. QR Pairing: Use `qrLogin()`
  b. Manually: Use `manualLogin()`
2. Payment
  Use `doPayment()`
3. Refund
  Use `doRefund()`
